  • Space Battle

    Unreal Engine 5.0
    Space Battle is a solo project using Unreal Engine 5.0. This is two player top down space battle game where you have to destroy the other players ship with in the round time. This project was made to use the Unreal's Paper System to make a 2D game.

  • Bet365 Challenge

    Unity Engine 2022.1.4f1
    This Project was a Duo project with Jakub Bas. In the challange we had to make a graphical simulation using real-time GPS coordinates stored in a JSON to desplay the position of each horse. this Project was made using Unity Engine because we had 6 hours to create this desplay for the horses. In this challange I worked on creating a script that would load data out of the JSON file and get the data ready to be passed to the horses so the position could be updated.

  • Rooted - Global Gam Jam 2023

    Unity Engine 2022.1.4f1
    This was a 2D top down puzzel game I made for the Global Game Jam 2023. This was made in Unity Engine using C# where I reamde a minigame from Club Pengiun. This uses a tile system which spawns the levels from JSON files with strings storing the tile data for the level which lets levels be easly modified. When the player is spawned it gets passed starting node which is used to check if a tile is walkable
